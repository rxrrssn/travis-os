# Governance Principles

## Canonical Reality

There is one canonical instance of:

- Person
- Organization
- Course
- Program
- Policy

Everything else references that reality.

## Do It Once, Reuse Everywhere

Information is created once and projected everywhere.

## People Change, Roles Persist

Authority belongs to roles and relationships, not individuals.

## Tenant Isolation

Tenant isolation is a security boundary.

## No Special Cases

No hardcoded Corveaux behavior.

No if tenant == "corveaux" logic.

Corveaux must operate using the same platform capabilities as every future institution.
